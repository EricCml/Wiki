# 实战wiki知识库系统

## Spring Boot + Vue3 前后端分离

### 项目简介

一个界面简约、功能完整的知识库系统。主要功能有用户登录、电子书管理、文档管理、富文本编辑、图片上传、文档点赞、报表展示等功能。
